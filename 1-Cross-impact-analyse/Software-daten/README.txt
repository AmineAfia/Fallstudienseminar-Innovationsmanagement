To reproduce the matrix and scenarios follow this steps:

1- install the Szeno-plan tool
2- create a folder to gather the generated files
3- in the tool: go to Datei-->neues szenario
4- navigate to the created folder (it still empty now)
5- give the name 111111111
6- copy the files from the reproduce-matrix-and-scenarios folder (from google drive)
7- you can see the matrix (cross-impact and konsistenz) and generate the scenarios
